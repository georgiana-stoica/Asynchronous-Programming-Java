# Asynchronous vs Concurrent Programming

**Concurrent** - a piece of code executed by more than one thread ( more things can be done at the same time ) \
**Synchronous** - a thread that launches a task ( that thread waits for its  completion ) \
**Asynchronous** - a task will be executed in some point in time in the future and that thread that creates the task will not wait for its completion \

Asynchronous may not be concurrent, and vice versa.
